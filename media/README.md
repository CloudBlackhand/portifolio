# Sua foto profissional

Coloque aqui a sua foto com um destes nomes (recomendado):

- `foto.jpg`
- `foto.png`

Ou qualquer `.jpg` / `.png` / `.webp` (o build usa a mais recente).

**Importante:** nao use arquivos `.crdownload` (download incompleto do navegador). Salve a imagem final na pasta antes do deploy.

O script `npm run sync-profile` copia para `public/profile/photo.jpg`, exibida na secao "Uma conversa clara sobre o seu projeto" na pagina inicial.
